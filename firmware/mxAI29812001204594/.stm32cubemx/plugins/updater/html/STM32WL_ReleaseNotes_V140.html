STM32CubeWL Firmware Package v1.4.0 / 29-October-2025
Main Changes
  • CMSIS/LL/HAL Drivers updates:
    • Maintenance release to include latest corrections including the following IP drivers: ADC, RCC, HSEM, I2C, SUBGHZ, RTC, TIM, LPTIM, UART, USART, FLASH, SPI, DMA
    • Fix: STM32CubeIDE projects location of the .size directive in startup code
    • Fix: ROM symbol addresses in template files to use code region aliases
    • Feature: Allow redefinition of the macro VECT_TAB_OFFSET externally from the IDE, makefile, or command line
    • Fix of MISRAC2012 Rules 13.4_b, 12.3, 10.4 and 8.13 violations
  • LoRaWAN Middleware updates:
    • Fix: rebootTimeReq is based on GPS epoch, convert to Unix epoch
    • Fix: LCTT Test BV_015_A for CHANNEL_PLAN_GROUP_AS923_1_JP_CH24_CH38_LBT test due to allowed data rate mismatch
    • Fix: Backoff Algorithm issue (patch from Semtech Github #1399)
    • Fix: Fixed typo in comments and channel plan name (From CH37 to CH33) for AS923_1 with LBT and DC
  • Security Middleware updates:
    • Migrate to MbedTLS V3.6.4
    • Migrate to STM32 Key Management Services V1.2.3
    • Migrate to STM32_Secure_Engine V2.5.4
    • Migrate to FreeRTOS V10.6.2
  • Projects updates:
    • SLA0044 Legal terms updated to latest revision (Rev 6 / October 2025)
    • Project alignment to STM32CubeMX V6.16.0
    • Project alignment to updated FreeRTOS and MbedTLS versions
  • Package updates:
    • SLA0048 Legal terms updated to latest revision (Rev 5 / October 2025)
