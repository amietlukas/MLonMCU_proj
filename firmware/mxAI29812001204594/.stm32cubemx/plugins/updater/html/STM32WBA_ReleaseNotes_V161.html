STM32CubeWBA Firmware Package V1.6.1 / 11-April-2025
Main Changes
Patch Release of STM32CubeWBA Firmware package supporting STM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA5Mxx, STM32WBA55xx, STM32WBA62xx, STM32WBA63xx, STM32WBA64xx and STM32WBA65xx devices
    Security
    • OEMuRoT update is supported
    • OEMuRoT swap mode is functional
    Connectivity
    • BLE:
      • Add a new API to switch from the default RF configuration without CTE support to a configuration supporting CTE
    • BLE Audio:
      • Scan Delegator supports broadcast Streams with 2 subgroups
    • Concurrent BLE Thread:
      • Stability improvement
    • Zigbee:
      • Zigbee Direct :
      • Enhancement of the ZigBee Direct ecosystem by adding a new Application
      • Expose mandatory BLE services requested by ZigbeeDirect to the Application
    Drivers
    • CMSIS Device Drivers supporting TM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA55xx, STM32WBA5Mxx, STM32WBA62xx, STM32WBA63xx, STM32WBA64xx and STM32WBA65xx devices
    • HAL/LL Drivers available for all peripherals
    • HAL: ADC, COMP, CORTEX, CRC, CRYP, DMA, EXTI, FLASH, GPIO, GTZC, HASH, HSEM, HCD, I2C, ICACHE, IRDA, IWDG, LPTIM, PKA, PWR, PCD, RAMCFG, RCC, RNG, RTC, SAI, SMARTCARD, SMBUS, SPI, TIM, TSC, UART, USB, USART, WWDG
    • LL : ADC, COMP, CRC, DMA, EXTI, GPIO, I2C, ICACHE, LPTIM, LPUART, PKA, PWR, RCC, RNG, RTC, SPI, TIM, USART, UTILS
    • Update HAL/LL drivers to include latest corrections and to support of STM32WBA6xx devices
    • BSP Drivers : Update STM32WBAxx_Nucleo, STM32WBA65I-DK1 and STM32WBA55G-DK1 BSP drivers to include latest corrections.
    • Add B-WBA5M-WPAN BSP drivers
