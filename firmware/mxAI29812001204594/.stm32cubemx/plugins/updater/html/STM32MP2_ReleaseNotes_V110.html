STM32CubeMP2 Firmware Package V1.1.0 / 07-Nov-2024
Main Changes
This is a **Maintenance release of STM32CubeMP2** Firmware Package v1.1.0 release
  • Added support for a new derivative of MP2x: MP23
    • Added CMSIS for all CPNs of this derivative.
    • Added BSP for a new Board STM32MP235F-DK (This board is not available, but a BSP is available which has been tested on STM32MP257F-DK board)
    • Added new examples, Applications and Demonstration projects for MP23.
    • ResourceManager updated to handle requests for MP23 resources.
  • MP25: Support added for Cortex-M0
    • CMSIS for all CPNs available.
    • OpenOCD scripts updated to support CM0+ project compilation.
    • Demonstration project running in production mode showing usage of LPUART
    • Cortex-M0 firmware is running in production mode.
    • Added script fw_cortex_m0.sh to load, start, stop the firmware.
    • This project can be compiled using Cmake or STM32CubeIDE.
    • Uses Open config of OSTL.
  • CMSIS updated to v5.9.0
  • MP25: Low Power modes demonstration.
    • Demo Cube Firmware project that is loaded and executed from the SRAM in production mode.
  • MP2x: Support added to encrypt the Cube Firmware and decrypt when loaded on M33.
  • MP2x: Updated Resource Manager
    • Support for multiple GPIO pins request/release.
    • Support to request RTC features as resources.
    • Bug Fixes, refactoring of codebase to allow addition of MP2x derivatives.
  • MP25: Support added for 16x16 package.
  • MP2x: Miscellaneous bug fixes in DMA, GPIO, RCC
  • Misra, MCUAstyle fixes.
  • General Code cleanup
