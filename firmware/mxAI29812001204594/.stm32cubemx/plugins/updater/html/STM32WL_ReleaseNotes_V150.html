STM32CubeWL Firmware Package v1.5.0 / 04-February-2026

Main Changes

  • CMSIS Core Drivers updates
    • Update to V5.9.0
  • CMSIS Device Drivers updates
    • Update to V1.4.0 to fix MISRAC-2012 rules violations
  • HAL/LL Drivers updates
    • Update the following HAL drivers to get the latest corrections: ADC, CRYP, SUBGHZ, SPI, I2S, TIM, RCC, RTC, UART, USART
    • Update SubGHz drivers to add new public APIs needed by the LoRAWAN-LBM stack
  • Middlewares updates
    • Update LoRaWAN middlewares to new LBM (Lora Basic Modem) stack
    • Update SubGHz_Phy to V1.4.0 supporting new LoRAWAN-LBM stack. All non-LoRA applications are supported
    • Update FatFS to version r0.12c_stm32cube_20240212 to include the latest bug fixes ensuring API backward compatibility
    • Update STM32_Key_Management_Services MW to V1.2.4
  • Projects updates
    • Remove all LoRaWAN projects based on LoRAWAN Mac-Node legacy middleware
    • Update all wireless applications to the latest toolchains versions
    • Update all wireless applications to use the new SubGHz_Phy middleware
    • Add the new LoRAWAN-LPBM based applications
      • LoRaWAN_Relay_LBM
      • LoRaWAN_End_Node_Relay_LBM
      • LoRaWAN_End_Node_LBM
    • Resolve compilation warnings in various projects when using newer GCC versions
    • Fix a regression in Sigfox applications
    • Update BFU/SBSFU applications to enforce post-build actions dependencies with EWARM v9.60.3
    • Update KMS applications to remove differences versus STM32CubeMX generated code
