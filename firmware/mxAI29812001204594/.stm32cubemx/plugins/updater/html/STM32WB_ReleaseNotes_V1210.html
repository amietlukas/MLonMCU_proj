STM32CubeWB Firmware Package V1.21.0 / 30-October-2024

Main Changes

    Maintenance Release for BLE, 802.15.4, Thread and Zigbee updates

  • BLE updates
    • Update to latest TCRL-2024.1.
    • Several bug fixes on Copro side.
  • MAC updates
    • Several minor bug fixes : Remove wrong Rx buffer allocation notification, Panid conflict management regression on 1.19 release.
  • Thread updates
    • OpenThread update to SHA-1: 4a0d6b7b6495a1e4e2e03ab7a4f8baa064df4a6e (June 11th 2024).
  • Zigbee updates
    • The default number of routing tables have been updated. Refer to the wiki page for more details :
      • https://wiki.st.com/stm32mcu/wiki/Connectivity:STM32WB_Zigbee_memory_requirements

    Maintenance Release for HAL/LL Drivers and Projects updates

  • HAL/LL Drivers updates
    • Update HAL/LL drivers to include latest corrections
      • Update of HAL ADC, QSPI, SPI, UART, USB drivers.
      • Update of LL ADC, SPI, USB drivers.
    • Refer to release notes for further details
