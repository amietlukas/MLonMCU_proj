STM32CubeU3 Firmware Package V1.3.0 / 04-February-2026

Main Changes

Official release of STM32CubeU3 FW package

Drivers

- CMSIS device drivers supporting STM32U3xx devices

- HAL/LL drivers available for all peripherals

- BSP drivers supporting NUCLEO-U3C5ZI-Q and NUCLEO-U385RG-Q boards

Middlewares

- MW ST CMSIS-RTOS ThreadX wrapper V1.4.0 support

- MW ST Filex V6.4.0 support

- MW ST Levelx V6.4.0 support

- MW ST Netxduo V6.4.0 support

- MW ST ThreadX V6.4.0 support

- MW ST USBX V6.4.0 support

- MW ST HSP V1.0.0 support

- MW mbed-crypto version V3.6.4 support

- MW mcuboot version V1.7.2 support

Projects

- Templates and Templates_LL projects based on HAL/LL APIs

- HAL and LL examples for IP peripherals

- HSP Applications

- Root of Trust (ROT) applications

- AzureRTOS 6.4.0 applications
