STM32CubeMX Software V6.16.0 / November-2025
Major release :
  • Added the support for new microcontrollers in the STM32WBA series
  • Added the support for new microcontrollers in the STM32N6 series
  • Added the support for new microcontrollers in the STM32H5 series
  • Added the support for new microcontrollers in the STM32WL3x product line
  • Added the support for new boards: NUCLEO-WL3RKB1 and NUCLEO-WL3RKB2
  • Implemented the Start From Board feature for NUCLEO-WL3RKB1 and NUCLEO-WL3RKB2
  • Released a new installer that natively supports Apple® silicon
  • Implemented the memory management tool for STM32N6
  • Enhanced the build process using CMake
  • Removed the user authentication feature
  • Migrated to SamacSys as a new CAD provider for STM32CubeMX
