STM32CubeMX Software V6.15.0 / July-2025
Major release :
  • Added the trusted domain support for STM32MP2 microprocessors (Arm®Cortex®-A35 and Cortex®-M33)
  • Introduced STARM-Clang, STMicroelectronics LLVM-based toolchain for Arm®
  • Improved the build process with CMake
  • Implemented the RIF feature with the STM32MP2 trusted domains
  • Added the pause and resume options to software package download
  • Added the IAR Embedded Workbench® for Arm® toolchain support for STM32MP13xx microprocessors
  • Implemented the pinout compatibility for STM32MP2 microprocessors
  • Implemented the wM-Bus PHY middleware for STM32WL33xx microcontrollers
