STM32CubeWBA Firmware Package V1.5.0 / 30-October-2024
Main Changes
Official Release of STM32CubeWBA Firmware package supporting STM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA5Mxx and STM32WBA55xx devices
    Security
    • Minor updates in security applications (SBSFU and TFM) and related scripts.
    Connectivity
    • External PA(+20dbm) is now supported on BLE using BLE_TransparentMode application.
    • New third-party MW support: FreeRTOS, mbedtls (mbedtls_alt)
    • Deployment of new directory structure using Common/WPAN directory for Connectivity Applications
    • HSE clock startup instabilities fixed with SW workaround (please refer to errata sheet)
    • OpenThread update to SHA-1: 4a0d6b7b6495a1e4e2e03ab7a4f8baa064df4a6e (June 11th 2024)
    • Zigbee Smart Energy support
    • 802.15.4 Antenna Diversity (Mac_802_15_4_Node_Antenna_Diversity application)
    • Add Support of WBA5M (CEB Module) (Applications under B-WBA5M-WPAN)
    • Advertising Coding Scheme Selection features added
    Drivers
    • CMSIS Device Drivers supporting STM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA5Mxx and STM32WBA55xx devices
    • HAL/LL Drivers available for all peripherals
    • BSP Drivers : Update STM32WBAxx_Nucleo and STM32WBA55G-DK1 BSP drivers to include latest corrections. Add B-WBA5M-WPAN BSP drivers
    Projects
    • - B-WBA5M-WPAN
      • Add new applications demonstrating BLE, Thread and Zigbee capabilities
    • - NUCLEO-WBA55CG
      • Update Applications demonstrating BLE capabilities
      • Update Applications demonstrating Zigbee capabilities
      • Update Applications demonstrating Thread capabilities
    • - NUCLEO-WBA52CG
      • Update Applications demonstrating SBSFU/TFM capabilities
    • - STM32WBA55G-DK1
      • Update applications demonstrating BLE, Zigbee capabilities
