STM32CubeL5 Firmware Package V1.6.0 / 29-October-2025

Maintenance release of STM32CubeL5 Firmware
  • General updates to fix known defects and implementation enhancements.
  • SLA0044 Legal terms updated to latest revision (Rev 6 / October 2025)
  • SLA0048 Legal terms updated to latest revision (Rev 5 / October 2025)
  • TFM Applications are now distributed directly by the [TrustedFirmware.org](https://www.trustedfirmware.org/) community and are no longer supported within the STM32CubeL5 package.
    • Consequently, the trustedfirmware Lib has been removed.
    • For more information on using TF-M on STM32xx please refer to following links and documentation.
    • All details and link available at [tf-m-user-guide](https://trustedfirmware-m.readthedocs.io/en/latest/platform/stm/)
  • Upgrade MbedTLS library to use V3.6.4 (ST modified 2025-07-01).
  • Upgrade USBPD Device library to use L5 V5.1.1.
  • Upgrade FreeRTOS library to use V10.6.2 (ST modified 2024-10-11).
  • Upgrade mcuboot library to use v1.7.2.38.
  • Upgrade GUI_INTERFACE library to use GUI_V2.3.0.
