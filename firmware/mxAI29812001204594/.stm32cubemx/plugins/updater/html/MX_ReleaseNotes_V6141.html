STM32CubeMX Software V6.14.1 / April-2025
Minor release :
Patch release to fix the following issues detected in STM32CubeMX 6.14.0:
  • Issue with SAI feature configuration tab in projects using STM32H7Rx/7Sx and STM32H7Bxxx microcontrollers
  • DCMI building problems in STM32N6 projects
  • Failure in generating the initialization instructions for projects using the DCMIPP feature in CSI mode
  • ULPI_Clock initialization issues for STM32F7 projects using the USB_OTG_HS feature
  • STM32WBA63xx project creation issues
  • stdio.h inclusion error for STM32U5 projects
  • Failure to create projects that activate the SDMMC feature with a basic application structure
  • CMake and makefile issues with STM32WB series projects when configuring the STM32_WPAN middleware
  • Hard fault during the debugging of STM32H5 projects with Arm® TrustZone® enabled and peripheral memory region setting turned on
