STM32CubeWBA Firmware Package V1.8.0 / 29-October-2025
Main Changes
Official Release of STM32CubeWBA Firmware package supporting STM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA5Mxx, STM32WBA55xx, STM32WBA62xx, STM32WBA63xx, STM32WBA64xx and STM32WBA65xx devices
  • SLA0044 Legal terms updated to latest revision (Rev 6 / October 2025)
  • SLA0048 Legal terms updated to latest revision (Rev 5 / October 2025)
  • Security
    • Migrated middleware to MbedTLS version 3.6.4 LTS and to mcuboot version 1.7.2.38
    • Enabled STM32CubeIDE users on Windows OS to run RoT .bat scripts
    • Enforced Python 3 usage in all .bat scripts for ROT Apps and Provisioning (Removed all usage of .dll or .exe files in ROT projects)
    • Update Full Secure project setup (OEMiROT_Appli)
    • Python and some python packages are required to execute ROT_AppliConfig utilities on Windows. Refer to Utilities/PC_Software/ROT_AppliConfig/README.md for more details.
  • Connectivity
    • 802.15.4 and Concurrent mode applications: reducing number of applications to simplify the offer.
    • BLE:
      • Support of BLE 6.0
      • Man-In-The-Middle (MITM) management is enhanced in the BLE stack to provide more flexibility and better interoperability.
      • Add BLE Host Stack events FIFO buffer mechanism
    • BLE Audio:
      • Increase the max number of supported BIS from 2 to 6
      • Add new data path ID to Codec Manager for raw data
    • Zigbee:
      • The Zigbee Direct stack precertification has been completed (based on ZUTH)
    • Thread:
      • Upgrade of OT stack to SHA-1: cbb04be4a64910a8ad160ae1aae2e34fc31031b9
  • Drivers
    • CMSIS Device Drivers supporting TM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA55xx, STM32WBA5Mxx, STM32WBA62xx, STM32WBA63xx, STM32WBA64xx and STM32WBA65xx devices
    • HAL/LL Drivers available for all peripherals
    • HAL: ADC, COMP, CORTEX, CRC, CRYP, DMA, EXTI, FLASH, GPIO, GTZC, HASH, HSEM, HCD, I2C, ICACHE, IRDA, IWDG, LPTIM, PKA, PWR, PCD, RAMCFG, RCC, RNG, RTC, SAI, SMARTCARD, SMBUS, SPI, TIM, TSC, UART, USB, USART, WWDG
    • LL : ADC, COMP, CRC, DMA, EXTI, GPIO, I2C, ICACHE, LPTIM, LPUART, PKA, PWR, RCC, RNG, RTC, SPI, TIM, USART, UTILS
    • Update HAL/LL drivers to include latest corrections and to support of STM32WBA6xx devices
    • BSP Drivers : Update STM32WBAxx_Nucleo, STM32WBA65I-DK1 and STM32WBA55G-DK1 BSP drivers to include latest corrections.
