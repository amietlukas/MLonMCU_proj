STM32CubeMX Software V6.17.0 / February-2026
Major release :
  • Added the support for new microcontrollers in the STM32WBA2x product lines, STM32U3 series, and STM32H5 series
  • Added the support for new boards: NUCLEO-U3C5ZI-Q, NUCLEO-H5E5ZJ, and NUCLEO-WBA25CE1
  • Added the Start From Board feature for NUCLEO-U3C5ZI-Q, NUCLEO-H5E5ZJ, and NUCLEO-WBA25CE1
  • Released the following new features:
    • PLAY on STM32H5 devices with 4 Megabytes of flash memory
    • HSP on STM32U3 devices with 2 Megabytes of flash memory
    • User notification service
  • Enhanced the memory management tool implementation for the STM32N6 series
  • Added support for the CMake toolchain with STM32MP2 products
  • Enhanced the build process using CMake
  • Added support for Cortex-A35 coprocessor management through the Cortex-M33 nonsecure project
