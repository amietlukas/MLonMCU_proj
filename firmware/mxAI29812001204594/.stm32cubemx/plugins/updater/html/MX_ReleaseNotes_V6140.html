STM32CubeMX Software V6.14.0 / February-2025
Major release :
  • Added the support for the STM32WBA6xxx microcontrollers and related boards
  • Added the support for USBX multi-instance on STM32WBA6xxx microcontrollers
  • Added the support for the memory management tool (MMT) with STM32WBA6xxx microcontrollers
  • Improved the user interface (UI) of the Compare Projects tool
  • Improved the build process with CMake
  • Removed the Secure Manager boot path
